## Обзор

**Kenzap Nesting** — это настольное приложение, которое раскладывает DXF-детали по листам с помощью передового алгоритма 2D-вложения, работающего локально на вашей машине. Перетащите чертежи, задайте количество, настройте отступы и гравировку, экспортируйте результат как готовый к производству DXF-файл &mdash; без облачных туда-сюда и без оплаты за задачу.

Приложение спроектировано с учётом реальной практики операторов ЧПУ, лазерной, плазменной и гидроабразивной резки: смешанная геометрия, многослойные файлы, реалистичные ограничения по отступам и поворотам, и гравировка, которая должна располагаться на отдельном слое.
