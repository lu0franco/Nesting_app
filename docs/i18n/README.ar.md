## نظرة عامة

**Kenzap Nesting** هو تطبيق سطح مكتب يقوم بترتيب قطع DXF على الألواح باستخدام خوارزمية تعشيق ثنائية الأبعاد متطورة تعمل محلياً على جهازك. اسحب رسوماتك، حدد الكميات، اضبط التباعد والنقش، ثم صدّر النتيجة كملف DXF جاهز للإنتاج &mdash; دون الحاجة إلى السحابة ودون رسوم لكل مهمة.

تم تصميمه للواقع العملي لمشغلي CNC والليزر والبلازما والنفاثة المائية: هندسة مختلطة، ملفات متعددة الطبقات، قيود تباعد ودوران حقيقية، ونقش يحتاج إلى طبقة مستقلة.
