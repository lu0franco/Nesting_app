## 概要

**Kenzap Nesting** は、最先端の 2D ネスティングアルゴリズムをローカルで実行し、DXF パーツをシート上に配置するデスクトップアプリケーションです。図面をドラッグ＆ドロップし、数量を設定し、間隔と彫刻を構成して、結果を本番用 DXF としてエクスポートできます &mdash; クラウドへの往復もジョブごとの課金もありません。

CNC、レーザー、プラズマ、ウォータージェットの現場の実用的な現実を念頭に設計されています: 混合ジオメトリ、マルチレイヤーファイル、現実的な間隔と回転の制約、そして専用レイヤーに配置すべき彫刻。
