## 概要

**Kenzap Nesting** は、ローカルで動作する最新の 2D ネスティングアルゴリズムを使って、DXF パーツをシート上に配置するデスクトップアプリです。図面をドラッグ＆ドロップし、数量や間隔、彫刻設定を調整して、本番用 DXF として書き出せます &mdash; クラウドへの往復も、ジョブごとの課金もありません。

CNC、レーザー、プラズマ、ウォータージェットの現場で実際に必要になる条件を前提に設計されています。混在するジオメトリ、マルチレイヤーファイル、現実的な間隔や回転制約、そして専用レイヤーに分けて出力したい彫刻にも対応します。
